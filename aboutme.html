Few things I have created

  • Jenny 5 robot: a multi-purpose, almost humanoid robot. More details here: jenny5.org
  • Optical computing with time delays: a simple way to solve NP-complete problems with delayed photons. More details here: Optical computing with time-delays.
  • Multi Expression Programming: a Genetic Programming variant encoding multiple solutions in the same chromosome and thus exploring the search space better than single-solution-per-chromosome techniques. More details here: mepx.org
  • MEPX - crossplatform software (runs on Windows, MacOS, Ubuntu) for data analysis (regression and classification). Based on Multi Expression Programming. More details here: mepx.org
  • Automatic generation of heuristics. I evolved heuristics better than NN and MST for the Travelling Salesman Problem. More details here: Evolve heuristics
  • Traceless Genetic Programming: - a very fast version of Genetic Programming. Does not store the program itself, rather it stores only the output. More details here: Traceless Genetic Programming
  • No Free Lunch proof: I gave a practical proof that functions for which random search is better than a Genetic Algorithm exist.
  • 3D Printizer - a 3D printer. Joint collaboration with Ciocan Ioan and Ciocan Andrei. More details here: Uzina 3D

Classes I have initiated

  • Intelligent Robots - 2006-2017. More than 100 different robotic projects have been created using Lego, IRobot, Festo, Lynxmotion, Intellibrainbot robots.
  • Unconventional computing - 2007-2017. Theoretical and practical examples with Optical, DNA, Quantum, chemical, waster based, etc computers.

Competitions I have organized
